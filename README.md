# Backend

Features:

- Auth Service

  - Authentication
  - Access control
  - Rate limiting
  - Permission

- API Service

  - Pagination
  - Sorting
  - Dynamic query
  - Full text search
  - Geolocation

#### Setup

- Create a `.env` file in the root directory
  - `NODE_ENV=development`
  - `DATABASE_URL=postgresql://user@localhost/db

#### Running:

- `npm install`
- `npm run dev`

#### Commands:

- Sync models with database :`npm run generate`
